## new-ledger



A ledger id must begin with a network name followed by a / and a dbid. Network names and dbids may only contain lowercase letters and numbers. In your request, you must specify db/id. If the network specified does not exist, it creates a new network. This request returns a command id, the request does not wait to ledger to be fully initialized before returning.These requests do not need to be signed.
