The history page is where you can find the block histoy of your dataset, starting at block 2 when the dataset ledger is initialized.
Each block card displays the block number, hash, and timestamp. When you click on the right chevron more block details surface, the user id who excuted the transaction,
the number of changes (assertions or retractions of data), the affected classes, and transaction id. The block history can also be toggled between DESC and ASC.
